今儿来说说boost 的 function

1 最简单的用法
看代码
`

#include

#include

int fun(int a, int b)
{
return a + b;
}
double fund(double a, double b)
{
return a － b;
}
int fun2(int a, int b)
{
return a - b;
}

int main()
{
boost::function i_i_i;
i_i_i = &fun;
std::cout << fun(1, 2) << “ “
<< i_i_i(1, 2) << std::endl;
// 1) 3 3
　　　i_i_i = &fund;
std::cout << fund(3.1, 4.1) << “ “
<< i_i_i(3.1, 4.1) << std::endl;
// 2)7.2 7
i_i_i = &fun2;
std::cout << fun2(5, 6) << “ “
<< i_i_i(5, 6) <<　std::endl;
// 3) -1 -1
return 0;
}
`
从上述代码可以看出其实function 是定义了一种类型，
这种类型的对象可以执行一类返回值是int 并且有且具有两个int 参数的函数.

从2)处的输出值可以推测，上面那句i_i_i=&fund…….（留待继续探究）

class X { public: int foo(int x, int b) { return a + b; } } boost::function<int (X*, int, int)> f; f = &X::foo; std::cout << f(1, 2) << std::endl; // 3
