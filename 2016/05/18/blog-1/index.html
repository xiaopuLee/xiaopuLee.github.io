首先参考了 http://www.jianshu.com/p/858ecf233db9 搭建了hexo
nodejs 的安装 是下载了源码， 然后参考 http://blog.csdn.net/awj3584/article/details/18401539 编译的
然后就是配置_config.yml里面， 原帖子中的有误？ 使用了如下配置成功
deploy:
type: git #部署类型, 本文使用Github
repo: #部署的仓库的SSH
branch: master #部署分支,一般使用master主分支
常用的hexo 命令
hexo init
hexo generate
hexo server 通过localhost查看
hexi deploy 挂载到github
都可以用类似于 hexo d这样简写的方式
